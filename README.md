# React Mega Menu

A React project which aims to be a responsive and accessible boilerplate Mega
Menu.

## Objective

- Responsive
- Accessible
- WCAG 2.1 AAA compliant
- W3C valid

## View demo

Visit:
[https://react-mega-menu.netlify.app](https://react-mega-menu.netlify.app)
